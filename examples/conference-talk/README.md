# example: conference-talk

A full-length conference talk — cover, section dividers, stepped reveals, a highlighted code slide, quote/fact/statement layouts, and speaker notes. A realistic template for a 20–30 minute talk.

## Run it

```bash
pnpm --filter @astro-slides/example-conference-talk dev
# or, from this directory:
astro-slides dev
```

## Export it

```bash
astro-slides export --format pdf   # also: png, pptx, html
```

The deck source is `content/decks/conference-talk/slides.mdx`.
