# example: marketing-pitch

A landing-page-quality product pitch on the Cosmic theme — FlexBlock feature and metric grids, statement and quote layouts, and a call-to-action closer.

## Run it

```bash
pnpm --filter @astro-slides/example-marketing-pitch dev
# or, from this directory:
astro-slides dev
```

## Export it

```bash
astro-slides export --format pdf   # also: png, pptx, html
```

The deck source is `content/decks/marketing-pitch/slides.mdx`.
