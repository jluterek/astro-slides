# example: technical-tutorial

A code-heavy hands-on tutorial — Shiki line highlights, click-stepped code, a Magic Move sequence, a two-column code walkthrough, and a Mermaid diagram.

## Run it

```bash
pnpm --filter @astro-slides/example-technical-tutorial dev
# or, from this directory:
astro-slides dev
```

## Export it

```bash
astro-slides export --format pdf   # also: png, pptx, html
```

The deck source is `content/decks/technical-tutorial/slides.mdx`.
