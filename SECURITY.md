# Security Policy

## Supported versions

astro-slides is pre-1.0. Security fixes land on the latest published release; there is no
long-term-support branch yet.

| Version | Supported |
| --- | --- |
| latest `0.x` | ✅ |
| older | ❌ |

## Reporting a vulnerability

**Please do not open a public GitHub issue for security vulnerabilities.**

Report privately by email to **james.luterek@gmail.com** with:

- a description of the vulnerability and its impact,
- steps to reproduce (a minimal deck or command is ideal), and
- any suggested remediation.

You can expect an acknowledgement within a few days. Once the issue is confirmed and a fix is
released, we're happy to credit you in the release notes unless you'd prefer to remain anonymous.

Please give us a reasonable window to release a fix before any public disclosure.
