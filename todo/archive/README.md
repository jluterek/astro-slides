# Archive

Completed phase folders live here, intact. Each one was moved from `todo/NN-phase/` to `todo/archive/NN-phase/` at the moment it was distilled.

**You should rarely read files in here.** The distilled summary at `docs/built/NN-phase-name.md` is the right place to look when you need to know what was built or decided. The archive is forensics — open it only when investigating the history of a specific feature.

See `todo/README.md` § *Completing a phase* for the archival workflow.
